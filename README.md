# Password Manager Application
A C# Console Application designed to manage and store passwords securely. 

## Features
* **OOP Principles:** Demonstrates encapsulation and class structures.
* **Data Management:** Handles user inputs and displays stored information.
* **C# Fundamentals:** Built using .NET Core/Framework.

## How to Run
1. Clone the repository.
2. Open the solution in Visual Studio 2022.
3. Press `F5` to run the application.
